# NoBlockAlowPx

Всем здравствуйте 👋 это бесплатный проект против блокировок, в котором будет собрано самые надёжные решения,проект будет обновляться.

Для ПК 🖥️:
Zapret(YouTube,Discord)
TG WS Proxy(Telegram)
Karing(All)

Для Смартфона на базе OC Android📱:
ByeByeDpi(YouTube,Discord)
TG WS Proxy(Telegram)
Karing(All)

Для чего эти приложения?
Zapret, ByeByeDpi, позволяет обходить DPI блокировки, но есть минус не меняется IP-Адрес.

TG WS Proxy(ПК,Android Смартфон), позволяет создать свой локальный прокси для телеграмма.

Karing(Доступен везде), позволяет обходить блокировки и сменять IP-Адрес.
